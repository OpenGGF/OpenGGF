## Probe Methodology

**Construction approach:** try zero-arg, then `(ObjectSpawn)`, `(ObjectSpawn, String)`, `(ObjectSpawn, ObjectServices)` constructors with a `StubObjectServices` (no ROM, no OpenGL) and a representative spawn at (0x100, 0x100).

**Diff scope:** all non-static, non-synthetic primitive/enum/String fields in the concrete class hierarchy, stopping at `AbstractObjectInstance` (whose base fields are managed by the base restore path and not re-tested here).

**Fields NOT diffed:** object references (not equality-comparable without a live session), `@RewindTransient`-annotated fields (intentionally excluded from capture), array/collection/record compound fields (require deep-equals beyond scope).

**No frame-stepping:** capture runs immediately after construction. Fields that only diverge after one or more `update()` calls are NOT detected by this probe.
